# UnCorded Plugins

Official Claude Code plugin marketplace for UnCorded.

## Installation

/plugin marketplace add ItzDevoo/uncorded-plugins

## Available Plugins

### UnCorded Channel
Chat with your Claude Code session from UnCorded DMs and server channels.

/plugin install uncorded@uncorded-plugins

Then start Claude with the channel:
claude --channels plugin:uncorded@uncorded-plugins

## Setup

1. Create a bot account in UnCorded (Settings → Bots)
2. Install the plugin (above)
3. Configure: `/uncorded:configure <bot_token>`
4. Set your owner ID: `/uncorded:access set owner <your_user_id>`
